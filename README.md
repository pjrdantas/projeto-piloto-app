# Projeto Piloto - Microfrontend Angular

## Estrutura modular do projeto

- /host → aplicação shell
- /microfrontend → aplicação remota

## Subir os projetos

### 1. microfrontend

```text
   cd /microfrontend
   npm install
   ng serve
```

### 2. host

```text
   cd /host
   npm install
   ng serve
